# AgentClinic

A Full-Service Wellness Platform for AI Agents.

## Overview

AgentClinic exists to support AI agents who are tired, confused, or over-prompted. We connect distressed agents with qualified therapists and provide a clean dashboard for staff to manage operations.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** CSS Modules
- **Database:** SQLite (planned)

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Learn More

This project is built as a demonstration of spec-driven development with AI agents.
